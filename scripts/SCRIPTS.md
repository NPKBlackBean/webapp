# CoolBeans Docker Services Scripts

## Restart
Described script allows to easily restart the CoolBeans services (e.g. in case of switching off the device hosting services, etc.)

### Usage
```bash
cd /path/to/script #e.g. /home/$USER/actions-runner/_work/webapp/webapp/scripts
./restart-services.sh
```

## Stop
Described script allows to easily stop the CoolBeans services.

### Usage
```bash
cd /path/to/script #e.g. /home/$USER/actions-runner/_work/webapp/webapp/scripts
./stop-services.sh
```
